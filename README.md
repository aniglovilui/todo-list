# Todo list project

Приложение для управления задачами. Реализовано на React, Vite и Tailwind CSS.

## Возможности

- Добавление, редактирование, удаление задач
- Фильтрация
- Автосохранение в LocalStorage
- Адаптивный UI
- Горячие клавиши (Escape для закрытия модалок)

## Технологии

- **React 18**
- **Vite**
- **Tailwind CSS**
- **Lucide React**
- **LocalStorage**
